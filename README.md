# Amigo Secreto

Um aplicativo web simples para organizar sorteios de Amigo Secreto.

## Funcionalidades
- Adicionar nomes de amigos a uma lista.
- Validar entrada para evitar nomes vazios.
- Exibir lista de amigos numerada.
- Sortear um amigo aleatoriamente.
- Limpar campo de entrada após adicionar.
- Exibir resultado do sorteio.

## Tecnologias
- **HTML**: Estrutura da página.
- **CSS**: Estilização (arquivo `style.css`).
- **JavaScript**: Lógica interativa (`app.js`).
- **Google Fonts**: Fontes Inter e Merriweather.

## Estrutura do Projeto
- `index.html`: Página principal.
- `style.css`: Estilos da interface.
- `app.js`: Lógica de adição, listagem e sorteio.
- `assets/`: Imagens (ícone de sorteio e banner).

## Como Usar
1. Clone o repositório:
  ```
   git clone <URL_DO_REPOSITORIO>
  ```

- Abra index.html em um navegador.

- Digite um nome no campo de entrada e clique em "Adicionar".

- Repita até adicionar todos os amigos.

- Clique em "Sortear amigo" para escolher um nome aleatoriamente.

## Limitações
 
- Não impede nomes duplicados.
- Não persiste dados entre sessões.
- Sem animações ou temas customizáveis.
